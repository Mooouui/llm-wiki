# Andrej Karpathy

AI researcher and former OpenAI founding member / Tesla AI director. In April 2026, he published the "LLM Wiki" idea file on GitHub Gist, describing how he shifted the majority of his LLM token usage from coding to building a personal knowledge base. The post received over 17 million views on X.

## Key Ideas

- **LLM as knowledge maintainer**: Karpathy treats the LLM as a "programmer" maintaining a Markdown "codebase" (the wiki).
- **Idea files over code**: He advocates sharing abstract concept documents that LLM agents instantiate, rather than sharing specific implementations.
- **Obsidian + Claude Code workflow**: His personal stack uses Obsidian as the frontend IDE and Claude Code as the backend agent.
