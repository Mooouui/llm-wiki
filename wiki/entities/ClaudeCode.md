# Claude Code

Anthropic's CLI-based LLM agent. In the LLM Wiki architecture, Claude Code serves as the primary "programmer" that maintains the wiki. It reads CLAUDE.md as its schema/control protocol, which defines how to ingest, query, and lint the wiki.
