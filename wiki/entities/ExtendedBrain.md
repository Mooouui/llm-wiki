# Extended Brain

Publication (Substack) that wrote "The Wiki That Writes Itself" — the deepest critique of Karpathy's LLM Wiki approach. Argues that LLM Wiki eliminates essential cognitive friction, using the concept of hormesis and Luhmann's Zettelkasten as counter-examples.

## Key Argument

LLM Wiki produces beautiful maps but the user never walks the territory. Recommends a hybrid: use LLM for scaffolding (linking, indexing, gap-finding) but keep synthesis in your own hands.
